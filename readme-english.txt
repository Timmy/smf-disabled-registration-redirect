[size=5][b]Disabled Registration Redirect[/b][/size]

[b][size=10pt]Compatibility[/size][/b]
SMF 2.0.x

[b][size=10pt]Description[/size][/b]
A mod that adds the possibility to redirect users to a custom URL when they try to register while registration is disabled.

[b][size=10pt]Installation Information[/size][/b]
The Package Manager should work in most cases. If you have any problems, you can try to [url=http://wiki.simplemachines.org/smf/Manual_installation_of_mods]install this mod manually[/url].
